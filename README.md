# College Notes by kaif

A simple web application to upload and share college notes.

## Features

*   View a list of available notes.
*   Download notes.
*   Upload new notes (password protected).

## Technologies Used

*   Node.js
*   Express
*   Multer
*   HTML/CSS
*   JavaScript

## Setup and Installation

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Start the server with `node server.js`.

## Usage

*   Open a web browser and go to `http://192.168.0.124:5500`.
*   To upload a file, use the admin panel and the password 'admin'.

## Folder Structure

*   `server.js`: The main server file.
*   `public/index.html`: The main HTML file.
*   `notes/`: The directory where uploaded notes are stored.
*   `package.json`: Project metadata and dependencies.
